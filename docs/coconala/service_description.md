## よくある質問（FAQ）

### Q1. 素材（写真やテキスト）がなくても依頼できますか？
**A.** はい、大丈夫です！フリー素材のご提案や、テキストのたたき台作成もお手伝いします。ただし、オリジナル素材をご用意いただいた方がより効果的なLPになります。

### Q2. 納品後の修正は可能ですか？
**A.** 各プランに無料修正回数が含まれています（梅1回、竹2回、松3回）。それ以降の修正は1回につき3,000円で承ります。レイアウトの大幅変更は別途お見積りとなります。

### Q3. サーバーやドメインの準備は必要ですか？
**A.** 納品はHTML/CSSファイル一式となりますので、公開にはサーバーとドメインが必要です。設定方法のご説明や、ご希望があれば設置代行（+5,000円）も承ります。

### Q4. WordPressへの組み込みは可能ですか？
**A.** 基本プランはHTML納品ですが、WordPress対応も可能です。詳細はご相談ください（別途お見積り）。

### Q5. キャンセルはできますか？
**A.** 制作開始前であれば全額返金いたします。制作開始後のキャンセルは、進捗に応じたお支払いをお願いしております。
